# OpenClaw

OpenClaw is the stateless Telegram bridge for TokenWatch.

## Summary

▪️ receives Telegram updates
▪️ resolves the workspace-scoped integration through TokenWatch
▪️ executes analytics and Copilot tools
▪️ sends formatted replies back to Telegram
